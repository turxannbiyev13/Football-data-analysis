# Football Data Analysis ⚽

## Layihənin Məqsədi
Bu layihə **Sabah FK** və **Qarabağ FK** komandalarının son oyun performanslarını analiz etmək üçün hazırlanıb. Analiz mövzuları:  
- Hücum və müdafiə effektivliyi (xG, şutlar)  
- Qol statistikaları və qələbə faizi  
- Ev və səfər oyunlarının müqayisəsi  
- Statistika testləri və korrelyasiya analizləri

- **Sabah FK:** Son 13 oyunda 12 qalib, 1 heç-heçə. Hücum aktiv, şut sayı çox, amma bəzən bitirici dəqiqlik az olur.
