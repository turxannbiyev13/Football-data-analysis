# Football Data Analysis ⚽

## Layihənin Məqsədi
Bu layihə **Sabah FK** və **Qarabağ FK** komandalarının son oyun performanslarını analiz etmək üçün hazırlanıb. Analiz mövzuları:  
- Hücum və müdafiə effektivliyi (xG, şutlar)  
- Qol statistikaları və qələbə faizi  
- Ev və səfər oyunlarının müqayisəsi  
- Statistika testləri və korrelyasiya analizləri  

---

## İstifadə olunan Texnologiyalar
- **Python**  
- Kitabxanalar: `pandas`, `numpy`, `matplotlib`, `seaborn`, `plotly`, `scipy`  

---

## Layihə Faylları
| Fayl | Təsvir |
|------|--------|
| `Football AZPL.xlsx` | Oyun məlumatları (xG, Shots, Goals və s.) |
| `Football_Analysis.ipynb` | Colab notebook: analiz və vizuallaşdırma |
| `README.md` | Layihə haqqında məlumat və istifadə təlimatları |

---

## Nəticələrin Qısa Təhlili
- **Sabah FK:** Son 13 oyunda 12 qalib, 1 heç-heçə. Hücum aktiv, şut sayı çox, amma bəzən bitirici dəqiqlik az olur.  
- **Qarabağ FK:** Daha az qalibiyyət, amma stabillik və keyfiyyətli bitiricilər. Ev və səfər oyunları performans fərqlidir.
